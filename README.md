# 📊 Motorola Mobile Sales Dashboard (Power BI)

An interactive Power BI dashboard built to analyze Motorola mobile sales performance across different cities, months, payment methods, brands, and mobile models.

---

## 📌 Project Overview

This dashboard helps businesses monitor sales performance and identify trends using interactive visualizations.

The report includes KPIs, geographic analysis, customer ratings, payment methods, monthly sales trends, and product-wise performance.

---

## 🚀 Features

- 📈 Total Sales KPI
- 📦 Total Quantity Sold
- 💳 Total Transactions
- ⭐ Customer Ratings Analysis
- 🗺️ Sales by City (Map Visualization)
- 📅 Monthly Sales Trend
- 📱 Sales by Mobile Model
- 💰 Transactions by Payment Method
- 📆 Sales by Day Name
- 📋 Brand-wise Summary Table
- 🎯 Interactive Filters (Slicers)

---

## 📊 Dashboard KPIs

- Total Sales
